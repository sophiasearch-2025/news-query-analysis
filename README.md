# Proyecto de Búsqueda de Noticias

Este repositorio contiene el subsistema de backend para la búsqueda y filtrado de noticias indexadas en Elasticsearch.

## Documentación

La documentación completa de este subsistema se encuentra organizada en la carpeta `docs/`.

* [Ver Documentación Principal](./docs/README.md)
* [Arquitectura](./docs/ARQUITECTURA.md)
* [Guía de Despliegue](./docs/DEPLOY.md)

## Inicio Rápido

```bash
# Instalar dependencias
npm install

# Ejecutar servidor
npm start
```
